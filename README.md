# react-native-map-navigation

Enhances React Native Maps with Realtime Navigation. Based on react-native-maps-navigation - with improvements.

## Installation

**React Native >= 0.49**

```bash
yarn add react-native-map-navigation
```

Make sure you link the module before building:

```bash
react-native link react-native-map-navigation
```
